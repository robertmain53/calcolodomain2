Authoritative data source & methodology

Primary reference: NIST/SEMATECH e-Handbook of Statistical Methods, Section on measures of center (Median), latest online edition. See the official resource at NIST e-Handbook. All computations strictly follow the formulas and definitions provided by this source.

The formula explained

\[ \textbf{Given ordered data } x_{(1)} \le \dots \le x_{(n)}: \] \[ \text{Median } \tilde{x} = \begin{cases} x_{(\frac{n+1}{2})}, & \text{if } n \text{ is odd} \\ \dfrac{x_{(\frac{n}{2})} + x_{(\frac{n}{2}+1)}}{2}, & \text{if } n \text{ is even} \end{cases} \] \[ \text{Tukey quartiles: } Q_1 = \text{median of } \{x_{(1)}, \dots, x_{(\lfloor n/2 \rfloor)}\},\quad Q_3 = \text{median of } \{x_{(\lceil n/2 \rceil+1)}, \dots, x_{(n)}\}. \] \[ \text{Weighted median: given pairs } (v_i, w_i),\; \tilde{x}_w = \min \left\{ v_i : \sum_{v_j \le v_i} w_j \ge \tfrac{1}{2}\sum_{k} w_k \right\}. \]

Glossary of variables

  • n: number of observations.
  • x(i): i-th order statistic (i-th smallest value).
  • Median (x̃): middle value (or mean of the two middles when n is even).
  • Q1, Q3: first and third quartiles (Tukey method by default).
  • Weighted median: median considering observation weights.
  • Five-number summary: min, Q1, median, Q3, max.

How it works: a step-by-step example

Input: 12, 18, 18, 19, 21, 24, 24, 28, 30 (n = 9).

  1. Order the data (already ordered).
  2. Since n is odd, median is \( x_{(\frac{9+1}{2})} = x_{(5)} = 21 \).
  3. Lower half: {12, 18, 18, 19} → Q1 = (18 + 18)/2 = 18.
  4. Upper half: {24, 24, 28, 30} → Q3 = (24 + 28)/2 = 26.
  5. Min = 12, Max = 30 → Five-number summary: 12, 18, 21, 26, 30.

Frequently asked questions

Which separators are supported?

Spaces, commas, semicolons, or new lines. Decimal commas like 12,5 are recognized.

What happens with text or blanks?

Non-numeric tokens are ignored. The validator warns if fewer than two valid numbers are found.

How are quartiles computed?

Default is Tukey’s method. You can switch to Exclusive or Inclusive percentile style for Q1 and Q3.

Is the weighted median the same as the 50th percentile?

Yes, but weights influence the cumulative distribution. We return the smallest value for which cumulative weight ≥ 50% of total.

Can I get the ordered list back?

Yes—expand “Ordered data & positions” to see indices and sorted values.

Why is the median robust?

Because it depends only on order, not magnitude. Outliers do not pull it the way they pull the mean.

Do you support grouped data?

This version focuses on raw and weighted data. For grouped data, use mid-class values with frequencies as weights.
