The formula explained

Line — slope–intercept: \( y = m x + b \)

y-intercept: \( y(0) = b \).   x-intercept (if \(m \neq 0\)): set \(y=0\Rightarrow m x + b = 0 \Rightarrow x = -\tfrac{b}{m}\).

Line — standard form: \( a x + b y + c = 0 \)

y-intercept (if \(b \neq 0\)): \( y = -\tfrac{c}{b} \).   x-intercept (if \(a \neq 0\)): \( x = -\tfrac{c}{a} \).

Two points \((x_1,y_1),(x_2,y_2)\): \( m = \dfrac{y_2 - y_1}{x_2 - x_1} \) (with \(x_2 \neq x_1\)), \( b = y_1 - m x_1 \).

Quadratic: \( y = a x^2 + b x + c \)

y-intercept: \( y(0) = c \). Discriminant \( \Delta = b^2 - 4ac \).

x-intercepts (real) when \( \Delta \ge 0 \): \( x = \dfrac{-b \pm \sqrt{\Delta}}{2a} \) with \( a \neq 0 \).

Glossary of variables

  • m — slope of a line.
  • b — y-intercept of a line \(y=mx+b\).
  • a,b,c — coefficients of line \(ax+by+c=0\) or quadratic \(ax^2+bx+c\).
  • x-intercepts — values of x where \(y=0\).
  • y-intercept — value of y at \(x=0\).
  • \(\Delta\) — quadratic discriminant \(b^2-4ac\).

How it works: a step-by-step example

Example 1 — line \(y=2x-6\)

y-intercept \(= b = -6\). For x-intercept: set \(0 = 2x - 6 \Rightarrow x = 3\).

Example 2 — quadratic \(y=x^2-5x+6\)

\(\Delta = (-5)^2 - 4(1)(6) = 25-24=1\). Roots \(x = \frac{5 \pm 1}{2}\Rightarrow x=2,\,3\). y-intercept \(= c = 6\).

Frequently asked questions

What if the line is vertical?

Use standard form with \(b=0\). The equation is \(ax+c=0 \Rightarrow x=-c/a\). There is no finite y-intercept.

What if the line is horizontal?

For \(m=0\), the line is \(y=b\). x-intercepts exist only if \(b=0\) (the x-axis itself).

How are complex roots handled?

If \(\Delta<0\) for a quadratic, there are no real x-intercepts; the calculator reports “no real intercepts”.

Rounding and display?

Results use up to the selected number of significant digits with trimmed trailing zeros to stay readable.

Can I derive the line from two points?

Yes—select “Two points”, enter \((x_1,y_1)\) and \((x_2,y_2)\) with \(x_1 \ne x_2\). The tool computes \(m\), \(b\), then the intercepts.
