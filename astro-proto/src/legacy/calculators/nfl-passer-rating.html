What is NFL passer rating?

NFL passer rating (often called “QB rating”) is the traditional statistic the league uses to summarize a quarterback’s passing efficiency. It combines completion percentage, yards per attempt, touchdown rate, and interception rate into a single number on a 0–158.3 scale.

This calculator uses the official NFL formula, so you can plug in box-score stats for a single game, a season, or an entire career and instantly see the corresponding passer rating.

NFL passer rating formula

The NFL formula uses five basic passing stats:

  • ATT – pass attempts
  • COMP – completions
  • YDS – passing yards
  • TD – touchdown passes
  • INT – interceptions

From these, four intermediate components are computed:

1. Accuracy component (a)

\[ a = \left(\frac{\text{COMP}}{\text{ATT}} - 0.3\right) \times 5 \]

2. Yardage component (b)

\[ b = \left(\frac{\text{YDS}}{\text{ATT}} - 3\right) \times 0.25 \]

3. Touchdown component (c)

\[ c = \frac{\text{TD}}{\text{ATT}} \times 20 \]

4. Interception component (d)

\[ d = 2.375 - \left(\frac{\text{INT}}{\text{ATT}} \times 25\right) \]

Each component is then clamped to the range 0 to 2.375:

\[ \text{component} = \min\bigl(2.375,\ \max(0,\ \text{component})\bigr) \]

Finally, the passer rating is:

\[ \text{Rating} = \frac{a + b + c + d}{6} \times 100 \]

The theoretical minimum is 0.0 and the theoretical maximum is 158.3, which occurs when all four components equal 2.375.

Worked example

Suppose a QB has the following stat line:

  • ATT = 35
  • COMP = 24
  • YDS = 275
  • TD = 2
  • INT = 1

First compute the per-attempt rates:

  • Completion % = 24 / 35 ≈ 68.6%
  • Yards per attempt = 275 / 35 ≈ 7.86
  • TD per attempt = 2 / 35 ≈ 0.0571
  • INT per attempt = 1 / 35 ≈ 0.0286

Now plug into the components:

  • \(a = ((24/35) - 0.3) \times 5 ≈ 1.93\)
  • \(b = ((275/35) - 3) \times 0.25 ≈ 1.21\)
  • \(c = (2/35) \times 20 ≈ 1.14\)
  • \(d = 2.375 - ((1/35) \times 25) ≈ 1.66\)

All four are already between 0 and 2.375, so no clamping is needed.

Sum and scale:

\[ \text{Rating} = \frac{1.93 + 1.21 + 1.14 + 1.66}{6} \times 100 \approx 99.0 \]

So this performance would earn a passer rating of about 99.0, which is comfortably above recent league averages.

How to interpret passer rating

Ranges vary by era, but for modern NFL seasons a rough guide is:

  • Below 75 – Poor
  • 75–90 – Below average
  • 90–100 – Solid / slightly above average
  • 100–110 – Very good
  • 110+ – Elite efficiency

For single games, ratings can swing much more wildly. A QB can post a rating near 0 with a few bad throws, or approach 158.3 in a highly efficient outing.

Limitations of passer rating

While passer rating is widely used, it has several important limitations:

  • Ignores sacks and sack yardage.
  • Ignores rushing (scrambles, designed runs, rushing TDs).
  • Does not account for down, distance, or game situation.
  • Does not adjust for opponent strength or supporting cast.
  • Weights each component in a fixed, somewhat arbitrary way.

More advanced metrics like ESPN Total QBR, EPA/play, or adjusted net yards per attempt (ANY/A) try to address some of these issues. But passer rating remains useful because it is simple, transparent, and historically consistent.

NFL passer rating FAQ

What is a good NFL passer rating?

In recent years, league-average passer rating has hovered in the low-to-mid 90s. As a rule of thumb, below 75 is poor, 75–90 is below average, 90–100 is solid, 100–110 is very good, and anything above 110 over a full season is elite.

What is the maximum NFL passer rating?

The theoretical maximum is 158.3. This happens when all four components of the formula reach their cap of 2.375. In practice, that requires a very high completion percentage, lots of yards per attempt, multiple touchdowns, and no interceptions.

Can passer rating be negative?

No. Each component is floored at 0 before being combined, so the overall rating cannot go below 0.0. Very poor games will simply produce ratings near zero.

Does this calculator work for season or career stats?

Yes. The NFL formula is based on rates (per attempt), so you can use game, season, or career totals. Just make sure all stats (attempts, completions, yards, TD, INT) are from the same sample.

How is passer rating different from QBR?

Passer rating is a simple, public formula using only passing box-score stats and scaled from 0 to 158.3. ESPN’s Total QBR is a proprietary 0–100 metric that incorporates play-by-play context, rushing, sacks, game situation, and opponent strength. They are designed for different purposes and are not directly comparable.
