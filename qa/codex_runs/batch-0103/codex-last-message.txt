        <h2 class="text-2xl font-semibold text-gray-800">How to Use This Calculator</h2>
        <p class="text-gray-600">Enter the hot and cold fluid inlet/outlet temperatures, then press Calculate. The tool computes the two temperature differences and applies the log mean formula to estimate the effective driving force for heat transfer.</p>
        <h2 class="text-2xl font-semibold text-gray-800">Methodology</h2>
        <div class="bg-gray-50 border border-gray-200 rounded-lg p-4 text-sm text-gray-700">
          <p>The calculator follows the standard LMTD approach for counterflow heat exchangers, deriving the average temperature difference from both ends of the shell-and-tube arrangement.</p>
          <p class="mt-2">Tutti i calcoli si basano rigorosamente sulle formule e sui dati forniti da questa fonte.</p>
        </div>
        <h3 class="text-xl font-semibold text-gray-800">Glossary of Terms</h3>
        <ul class="list-disc pl-5 text-gray-700 space-y-2">
          <li><strong>Hot Inlet Temperature:</strong> hot fluid temperature entering the heat exchanger.</li>
          <li><strong>Hot Outlet Temperature:</strong> hot fluid temperature leaving the exchanger.</li>
          <li><strong>Cold Inlet Temperature:</strong> cold fluid temperature entering the exchanger.</li>
          <li><strong>Cold Outlet Temperature:</strong> cold fluid temperature exiting the exchanger.</li>
          <li><strong>LMTD:</strong> Log Mean Temperature Difference between the two streams.</li>
        </ul>
        <h3 class="text-xl font-semibold text-gray-800">Step-by-Step Example</h3>
        <p class="text-gray-600">Take a heat exchanger where the hot fluid enters at 150°C and leaves at 100°C while the cold fluid flows from 50°C to 80°C. The calculator evaluates ΔT₁ = 150 − 80 = 70°C and ΔT₂ = 100 − 50 = 50°C, then computes LMTD ≈ 59.44°C.</p>
        <h3 class="text-xl font-semibold text-gray-800">Frequently Asked Questions</h3>
        <div class="space-y-4 text-sm text-gray-700">
          <div>
            <h4 class="font-semibold text-gray-800">What is the purpose of the LMTD method?</h4>
            <p>The LMTD method determines the average temperature difference between two fluids in a heat exchanger, which is essential for calculating heat transfer rates.</p>
          </div>
          <div>
            <h4 class="font-semibold text-gray-800">Why is LMTD important?</h4>
            <p>LMTD provides a more accurate representation of the temperature driving force than a simple arithmetic mean.</p>
          </div>
          <div>
            <h4 class="font-semibold text-gray-800">Can LMTD be applied to all types of heat exchangers?</h4>
            <p>Yes, the concept is universal; you only need to adjust for flow arrangement if it differs.</p>
          </div>
          <div>
            <h4 class="font-semibold text-gray-800">What factors affect the LMTD?</h4>
            <p>The inlet and outlet temperatures of both fluids and the direction of flow impact the LMTD.</p>
          </div>
          <div>
            <h4 class="font-semibold text-gray-800">How does LMTD differ from arithmetic mean temperature difference?</h4>
            <p>LMTD accounts for the logarithmic pattern of temperature change, offering a precise driving-force estimate.</p>
          </div>
        </div>

        <details id="formulaDetails">
          <summary class="cursor-pointer font-bold text-gray-800">Formulas</summary>
          <div class="mt-3 p-3 bg-white border border-gray-200 rounded-lg text-sm space-y-2">
            <p class="font-semibold">Log Mean Temperature Difference</p>
            <div class="formula-box">
              <p>\( LMTD = \frac{\Delta T_1 - \Delta T_2}{\ln(\Delta T_1/\Delta T_2)} \)</p>
            </div>
            <p class="text-gray-600">Where:</p>
            <ul class="list-disc pl-5 text-gray-600 text-sm space-y-1">
              <li>ΔT₁ = Hot inlet − Cold outlet</li>
              <li>ΔT₂ = Hot outlet − Cold inlet</li>
            </ul>
          </div>
        </details>

    <script>
      (function () {
        'use strict';

        const debounce = (fn, delay = 100) => {
          let timeout;
          return (...args) => {
            clearTimeout(timeout);
            timeout = setTimeout(() => fn(...args), delay);
          };
        };

        const round2 = (value) => {
          if (!Number.isFinite(value)) return 0;
          return Math.round((value + Number.EPSILON) * 100) / 100;
        };

        const els = {
          hotInlet: document.getElementById('hotInlet'),
          hotOutlet: document.getElementById('hotOutlet'),
          coldInlet: document.getElementById('coldInlet'),
          coldOutlet: document.getElementById('coldOutlet'),
          lmtdValue: document.getElementById('lmtdValue'),
          deltaT1Value: document.getElementById('deltaT1Value'),
          deltaT2Value: document.getElementById('deltaT2Value'),
          ratioValue: document.getElementById('ratioValue'),
          lmtdNote: document.getElementById('lmtdNote'),
          errorBox: document.getElementById('errorBox'),
          calcBtn: document.getElementById('calcBtn'),
          resetBtn: document.getElementById('resetBtn')
        };

        const defaults = {
          hotInlet: 150,
          hotOutlet: 100,
          coldInlet: 50,
          coldOutlet: 80
        };

        function parseInputs() {
          return {
            hotInlet: parseFloat(els.hotInlet.value),
            hotOutlet: parseFloat(els.hotOutlet.value),
            coldInlet: parseFloat(els.coldInlet.value),
            coldOutlet: parseFloat(els.coldOutlet.value)
          };
        }

        function validate(inputs) {
          const errors = [];
          if (!Number.isFinite(inputs.hotInlet)) errors.push('Hot inlet temperature must be a number.');
          if (!Number.isFinite(inputs.hotOutlet)) errors.push('Hot outlet temperature must be a number.');
          if (!Number.isFinite(inputs.coldInlet)) errors.push('Cold inlet temperature must be a number.');
          if (!Number.isFinite(inputs.coldOutlet)) errors.push('Cold outlet temperature must be a number.');
          if (Number.isFinite(inputs.hotInlet) && Number.isFinite(inputs.coldOutlet) && inputs.hotInlet <= inputs.coldOutlet) {
            errors.push('Hot inlet must be warmer than the cold outlet.');
          }
          if (Number.isFinite(inputs.hotOutlet) && Number.isFinite(inputs.coldInlet) && inputs.hotOutlet <= inputs.coldInlet) {
            errors.push('Hot outlet must stay warmer than the cold inlet.');
          }
          return { ok: errors.length === 0, errors };
        }

        function compute(inputs) {
          const deltaT1 = inputs.hotInlet - inputs.coldOutlet;
          const deltaT2 = inputs.hotOutlet - inputs.coldInlet;
          if (!(deltaT1 > 0) || !(deltaT2 > 0)) return null;
          if (!Number.isFinite(deltaT1) || !Number.isFinite(deltaT2)) return null;
          const ratio = deltaT1 / deltaT2;
          let lmtd;
          if (Math.abs(deltaT1 - deltaT2) < 1e-9) {
            lmtd = deltaT1;
          } else {
            const logValue = Math.log(ratio);
            if (!Number.isFinite(logValue) || logValue === 0) return null;
            lmtd = (deltaT1 - deltaT2) / logValue;
          }
          if (!Number.isFinite(lmtd)) return null;
          return { deltaT1, deltaT2, ratio, lmtd };
        }

        function format(outputs) {
          return {
            lmtd: round2(outputs.lmtd).toFixed(2),
            deltaT1: round2(outputs.deltaT1).toFixed(2),
            deltaT2: round2(outputs.deltaT2).toFixed(2),
            ratio: round2(outputs.ratio).toFixed(3)
          };
        }

        function render(formatted = null, errors = []) {
          if (errors.length) {
            els.errorBox.style.display = 'block';
            els.errorBox.textContent = errors.join(' ');
            els.lmtdValue.textContent = '--';
            els.deltaT1Value.textContent = '--';
            els.deltaT2Value.textContent = '--';
            els.ratioValue.textContent = '--';
            els.lmtdNote.textContent = 'Fix the inputs above to get the LMTD.';
            return;
          }
          els.errorBox.style.display = 'none';
          els.errorBox.textContent = '';
          els.lmtdValue.textContent = formatted.lmtd;
          els.deltaT1Value.textContent = formatted.deltaT1;
          els.deltaT2Value.textContent = formatted.deltaT2;
          els.ratioValue.textContent = formatted.ratio;
          els.lmtdNote.textContent = 'Calculated for a counterflow configuration.';
        }

        function update() {
          const inputs = parseInputs();
          const validation = validate(inputs);
          if (!validation.ok) {
            render(null, validation.errors);
            return;
          }
          const outputs = compute(inputs);
          if (!outputs) {
            render(null, ['Unable to calculate the LMTD with the supplied temperatures.']);
            return;
          }
          render(format(outputs));
        }

        const debouncedUpdate = debounce(update, 100);
        [els.hotInlet, els.hotOutlet, els.coldInlet, els.coldOutlet].forEach((input) => {
          input.addEventListener('input', debouncedUpdate);
          input.addEventListener('change', debouncedUpdate);
        });

        els.calcBtn.addEventListener('click', update);
        els.resetBtn.addEventListener('click', () => {
          els.hotInlet.value = defaults.hotInlet;
          els.hotOutlet.value = defaults.hotOutlet;
          els.coldInlet.value = defaults.coldInlet;
          els.coldOutlet.value = defaults.coldOutlet;
          update();
        });

        update();
      })();
    </script>
