        <h2 class="text-2xl font-semibold">How to Use This Calculator</h2>
        <p>Set the covariance and market variance numbers for the security and benchmark you are analyzing. Click the Calculate button to refresh the beta value, then consider the interpretation and range a quick snapshot of market sensitivity.</p>
        <div class="bg-gray-50 border border-gray-200 rounded-lg p-4">
          <h3 class="text-xl font-semibold">Methodology</h3>
          <p>The calculator divides the covariance between the security and the market by the market variance, matching the definition used in finance textbooks. Results assume a stable market variance: update inputs if the benchmark volatility changes.</p>
        </div>
        <div>
          <h3 class="text-xl font-semibold">Data Source and Context</h3>
          <p>All calculations are based on standard financial analysis methodologies. Ensure accuracy by pulling official return series and variance estimates from a trusted market data provider before entering numbers.</p>
        </div>
        <div>
          <h3 class="text-xl font-semibold">Glossary of Terms</h3>
          <ul class="list-disc pl-5 space-y-1 text-gray-700">
            <li><strong>Covariance:</strong> A measure of how two return series move together. Positive covariance means the asset and market move in tandem.</li>
            <li><strong>Variance:</strong> The dispersion of returns for the benchmark. Use market variance to understand how volatile the reference index is.</li>
            <li><strong>Beta:</strong> The normalized ratio showing the asset's sensitivity to the benchmark. Beta of 1 tracks the market, greater than 1 is more volatile, and negative means an inverse relationship.</li>
          </ul>
        </div>
        <div>
          <h3 class="text-xl font-semibold">Illustrative Example</h3>
          <p>For instance, if the covariance between a stock and the market is 0.03 and the market variance is 0.01, the beta would be:</p>
          <p class="font-mono text-sm bg-gray-900 text-white rounded-lg inline-block px-3 py-2">Beta = 0.03 / 0.01 = 3.0</p>
        </div>
        <div>
          <h2 class="text-2xl font-semibold mt-6">Frequently Asked Questions</h2>
          <div>
            <h3 class="text-lg font-semibold">What is a Beta Calculator?</h3>
            <p>A beta calculator is a tool used to measure the volatility or systematic risk of a financial security or portfolio compared to the market as a whole.</p>
          </div>
          <div>
            <h3 class="text-lg font-semibold">How do I interpret a beta value?</h3>
            <p>A beta greater than 1 indicates the security is more volatile than the market, while a beta less than 1 indicates it is less volatile.</p>
          </div>
          <div>
            <h3 class="text-lg font-semibold">Why is beta important?</h3>
            <p>Investors use beta to understand a stock's market risk compared to the overall market, aiding portfolio diversification strategies.</p>
          </div>
          <div>
            <h3 class="text-lg font-semibold">What is the covariance in finance?</h3>
            <p>Covariance is a statistical measure of the directional relationship between the returns on two assets.</p>
          </div>
          <div>
            <h3 class="text-lg font-semibold">Can beta be negative?</h3>
            <p>Yes, a negative beta indicates an inverse relationship with the market, suggesting the asset tends to move opposite of the benchmark.</p>
          </div>

          return { beta };
        }

        function format(outputs, inputs) {
          const safeBeta = round(outputs.beta, 2);
          return {
            betaDisplay: safeBeta.toFixed(2),
            interpretation: classifyBeta(safeBeta),
            rangeLabel: describeRange(safeBeta),
            synopsis: `Based on covariance ${formatValue(inputs.covariance)} and variance ${formatValue(inputs.marketVariance)}.`
          };
        }

        function render(formatted, errors) {
          if (errors && errors.length > 0) {
            els.errorBox.style.display = 'block';
            els.errorBox.textContent = errors.join(' ');
          } else {
            els.errorBox.style.display = 'none';
            els.errorBox.textContent = '';
          }

          if (!formatted) {
            els.betaValue.textContent = '—';
            els.betaInterpretation.textContent = '—';
            els.betaRange.textContent = '—';
            els.betaSynopsis.textContent = 'Enter covariance and market variance, then click Calculate.';
            return;
          }

          els.betaValue.textContent = formatted.betaDisplay;
          els.betaInterpretation.textContent = formatted.interpretation;
          els.betaRange.textContent = formatted.rangeLabel;
          els.betaSynopsis.textContent = formatted.synopsis;
        }

        function update() {
          const inputs = parseInputs();
          const validation = validate(inputs);
          if (!validation.ok) {
            render(null, validation.errors);
            return;
          }

          const outputs = compute(inputs);
          if (!outputs) {
            render(null, ['Calculation failed. Please adjust your inputs.']);
            return;
          }

          const formatted = format(outputs, inputs);
          render(formatted, []);
        }

        els.calcBtn.addEventListener('click', update);
        els.resetBtn.addEventListener('click', () => {
          els.covariance.value = defaultValues.covariance;
          els.marketVariance.value = defaultValues.marketVariance;
          update();
        });

        const debouncedUpdate = debounce(update, 100);
        document.querySelectorAll('#inputsCard input')
          .forEach((el) => {
            el.addEventListener('input', debouncedUpdate);
            el.addEventListener('change', debouncedUpdate);
          });

        update();
      })();
    </script>
</body></html>
