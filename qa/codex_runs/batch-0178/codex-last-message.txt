          <section class="card content-card space-y-6" id="howToUse" aria-label="How to use and methodology">
            <h2 class="text-2xl font-semibold">Full original guide (expanded)</h2>
            <p>The BOD Calculator helps environmental engineers estimate the oxygen demand imposed on receiving waters by a wastewater sample. Enter your dissolved oxygen readings and the sample volume to see how much organic matter is consuming oxygen over the incubation period.</p>
            <h3 class="text-xl font-semibold mt-4">How to use this calculator</h3>
            <p>Begin by measuring the dissolved oxygen before putting the sample into a sealed bottle and again after the incubation period. Confirm that the final reading is less than or equal to the initial reading, then provide the exact volume of the sample analyzed in liters. Tap Calculate to instantly see the BOD value rounded to two decimal places.</p>
            <h3 class="text-xl font-semibold mt-6">Methodology</h3>
            <div class="optional-section">
              <p>All computations follow the procedures in <em>Standard Methods for the Examination of Water and Wastewater</em> and other environmental engineering references. The calculator subtracts the final dissolved oxygen from the initial reading, divides by the sample volume, and applies two-decimal rounding so the output aligns with laboratory reporting standards.</p>
            </div>
            <h3 class="text-xl font-semibold mt-6">Glossary of variables</h3>
            <ul class="list-disc pl-5 space-y-1 text-gray-700">
              <li><strong>Initial DO:</strong> Dissolved oxygen in the sample before incubation.</li>
              <li><strong>Final DO:</strong> Dissolved oxygen after the incubation period.</li>
              <li><strong>Sample Volume:</strong> Volume of the test sample in liters.</li>
              <li><strong>BOD:</strong> The biochemical oxygen demand in mg/L.</li>
            </ul>
            <h3 class="text-xl font-semibold mt-6">How it works</h3>
            <p>For example, a sample with initial DO 9 mg/L and final DO 3 mg/L in a 0.50 L bottle produces 12.00 mg/L of BOD. The calculator applies the formula in the Formulas section and displays the result alongside the input readings so you can verify measurement quality.</p>
            <h3 class="text-xl font-semibold mt-6">Frequently Asked Questions (FAQ)</h3>
            <div class="faq-item">
              <h4 class="text-lg font-semibold">What is Biochemical Oxygen Demand (BOD)?</h4>
              <p>BOD quantifies the amount of dissolved oxygen consumed by microorganisms while degrading organic matter under aerobic conditions.</p>
            </div>
            <div class="faq-item">
              <h4 class="text-lg font-semibold">Why is measuring BOD important?</h4>
              <p>BOD helps regulators and engineers assess the impact of wastewater discharges on dissolved oxygen levels in rivers and lakes.</p>
            </div>
            <div class="faq-item">
              <h4 class="text-lg font-semibold">How do I ensure accurate BOD measurements?</h4>
              <p>Follow standard laboratory procedures, keep samples sealed, and record DO readings carefully before and after incubation.</p>
            </div>
            <div class="faq-item">
              <h4 class="text-lg font-semibold">What factors can affect BOD results?</h4>
              <p>Temperature, presence of toxic substances, and delay between sampling and incubation are common sources of variation.</p>
            </div>
            <div class="faq-item">
              <h4 class="text-lg font-semibold">Can BOD ever be zero?</h4>
              <p>Yes; if the sample contains no biodegradable organics, the initial and final DO readings match, yielding zero BOD.</p>
            </div>

              <div>
                <p class="mb-2"><strong>Fixed formula:</strong></p>
                <div class="formula-box">
                  <p>\( \text{BOD} = \frac{\text{Initial DO} - \text{Final DO}}{\text{Volume}} \)</p>
                  <p class="mt-3">Example: \( \frac{9 - 3}{0.5} = 12 \ \text{mg/L} \)</p>
                </div>
                <ul class="list-disc pl-5 mt-3 text-gray-600">
                  <li>Initial DO and Final DO are in mg/L.</li>
                  <li>Volume is in liters.</li>
                  <li>All values are rounded to two decimals for reporting.</li>
                </ul>

      (function () {
        'use strict';

        var debounce = function (fn, delay) {
          if (delay === void 0) { delay = 100; }
          var timer;
          return function () {
            var args = [];
            for (var _i = 0; _i < arguments.length; _i++) {
              args[_i] = arguments[_i];
            }
            clearTimeout(timer);
            timer = setTimeout(function () { fn.apply(void 0, args); }, delay);
          };
        };

        var round = function (value, decimals) {
          if (decimals === void 0) {
            decimals = 2;
          }
          if (!Number.isFinite(value)) { return 0; }
          var factor = Math.pow(10, decimals);
          return Math.round((value + Number.EPSILON) * factor) / factor;
        };

        var els = {
          initialDO: document.getElementById('initialDO'),
          finalDO: document.getElementById('finalDO'),
          volume: document.getElementById('volume'),
          bodValue: document.getElementById('bodValue'),
          initialValue: document.getElementById('initialValue'),
          finalValue: document.getElementById('finalValue'),
          deltaValue: document.getElementById('deltaValue'),
          volumeValue: document.getElementById('volumeValue'),
          errorBox: document.getElementById('errorBox'),
          calcBtn: document.getElementById('calcBtn'),
          resetBtn: document.getElementById('resetBtn')
        };

        var defaults = {
          initialDO: '9',
          finalDO: '3',
          volume: '0.5'
        };

        function parseInputs() {
          var parse = function (value) {
            var num = parseFloat(value);
            return Number.isFinite(num) ? num : null;
          };
          return {
            initialDO: parse(els.initialDO.value),
            finalDO: parse(els.finalDO.value),
            volume: parse(els.volume.value)
          };
        }

        function validate(inputs) {
          var errors = [];
          if (inputs.initialDO === null) { errors.push('Initial DO must be a numeric value.'); }
          if (inputs.finalDO === null) { errors.push('Final DO must be a numeric value.'); }
          if (inputs.volume === null) { errors.push('Sample volume must be a numeric value.'); }
          if (Number.isFinite(inputs.volume) && inputs.volume <= 0) {
            errors.push('Sample volume must be greater than zero.');
          }
          return { ok: errors.length === 0, errors: errors };
        }

        function compute(inputs) {
          var delta = inputs.initialDO - inputs.finalDO;
          var bod = delta / inputs.volume;
          if (!Number.isFinite(bod)) { return null; }
          return { bod: bod, delta: delta };
        }

        function format(outputs, inputs) {
          var formatValue = function (value, decimals) {
            if (decimals === void 0) { decimals = 2; }
            if (!Number.isFinite(value)) { return null; }
            return round(value, decimals).toFixed(decimals);
          };
          var formatted = {
            bod: formatValue(outputs.bod, 2),
            delta: formatValue(outputs.delta, 2),
            initial: formatValue(inputs.initialDO, 2),
            final: formatValue(inputs.finalDO, 2),
            volume: formatValue(inputs.volume, 3)
          };
          return {
            bod: formatted.bod ? formatted.bod + ' mg/L' : '—',
            delta: formatted.delta ? formatted.delta + ' mg/L' : '—',
            initial: formatted.initial ? formatted.initial + ' mg/L' : '—',
            final: formatted.final ? formatted.final + ' mg/L' : '—',
            volume: formatted.volume ? formatted.volume + ' L' : '—'
          };
        }

        function render(formatted, errors) {
          if (errors && errors.length) {
            els.errorBox.style.display = 'block';
            els.errorBox.textContent = errors.join(' ');
          } else {
            els.errorBox.style.display = 'none';
            els.errorBox.textContent = '';
          }
          els.bodValue.textContent = formatted.bod;
          els.initialValue.textContent = formatted.initial;
          els.finalValue.textContent = formatted.final;
          els.deltaValue.textContent = formatted.delta;
          els.volumeValue.textContent = formatted.volume;
        }

        function update() {
          var inputs = parseInputs();
          var validation = validate(inputs);
          if (!validation.ok) {
            render({ bod: '—', initial: '—', final: '—', delta: '—', volume: '—' }, validation.errors);
            return;
          }
          var outputs = compute(inputs);
          if (!outputs) {
            render({ bod: '—', initial: '—', final: '—', delta: '—', volume: '—' }, ['Calculation failed for the provided inputs.']);
            return;
          }
          var formatted = format(outputs, inputs);
          render(formatted, []);
        }

        function reset() {
          els.initialDO.value = defaults.initialDO;
          els.finalDO.value = defaults.finalDO;
          els.volume.value = defaults.volume;
          update();
        }

        els.calcBtn.addEventListener('click', update);
        els.resetBtn.addEventListener('click', reset);

        var debouncedUpdate = debounce(update, 100);
        document.querySelectorAll('#inputsCard input')
          .forEach(function (input) {
            input.addEventListener('input', debouncedUpdate);
            input.addEventListener('change', debouncedUpdate);
          });

        reset();
      })();
