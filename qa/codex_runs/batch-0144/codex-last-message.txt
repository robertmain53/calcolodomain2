        <section id="howToUse" class="card content-card" aria-label="How to use and methodology">
          <h2 class="text-2xl font-semibold">How to Use This Calculator</h2>
          <p>This professional-grade Bernoulli equation calculator helps engineers and students analyze steady, incompressible flow between two points. Provide known pressures, velocities, elevations, density, gravity, and any head loss. Pick the unknown variable from the dropdown and click Calculate to see the solved value plus converted outputs.</p>
          <p class="text-sm text-gray-600">Values update automatically with debounced input changes, and the solver enforces a consistent unit system internally to avoid rounding drift.</p>

          <h2 class="text-2xl font-semibold">Methodology</h2>
          <p>The tool assembles Bernoulli's energy equation between Point 1 and Point 2: <em>P1/(ρg) + v1²/(2g) + z1 = P2/(ρg) + v2²/(2g) + z2 + hL</em>. The selected unknown is isolated algebraically, so the equilibrium always balances the sum of pressure, velocity, gravitational, and head loss contributions.</p>
          <p>Once the unknown is determined, the engine recomputes both sides of the energy balance to report the resulting energy heads and pressure difference. All intermediate calculations run in double precision and convert back to the units you selected for display.</p>

          <div class="details-panel">
            <h3 class="text-lg font-semibold">Data Source and Limits</h3>
            <p><strong>Authoritative Source:</strong> Munson, Young and Okiishi’s <em>Fundamentals of Fluid Mechanics</em>, 8th Edition (2016), Wiley. Calculations follow the textbook relations for steady, incompressible, inviscid flow with optional head loss.</p>
            <p>The calculator assumes consistent units across both points. Input pressure references (absolute or gauge) must match. Use the head loss field to bundle major and minor losses computed separately.</p>
          </div>

          <section aria-label="Glossary">
            <h3 class="text-lg font-semibold">Glossary</h3>
            <ul class="list-disc pl-5 space-y-1 text-gray-700">
              <li>P1, P2 — Pressure at point 1 and 2 (Pa, kPa, MPa, bar, or psi)</li>
              <li>v1, v2 — Average velocity at point 1 and 2 (m/s or ft/s)</li>
              <li>z1, z2 — Elevation head at point 1 and 2 (m or ft)</li>
              <li>ρ — Fluid density (kg/m³ or slug/ft³)</li>
              <li>g — Gravitational acceleration (m/s² or ft/s²)</li>
              <li>hL — Head loss between points (m or ft)</li>
              <li>Heads — Pressure head P/(ρg), velocity head v²/(2g), elevation head z</li>
            </ul>
          </section>

          <section aria-label="Worked example">
            <h3 class="text-lg font-semibold">Worked example</h3>
            <p>Given water (ρ = 1000 kg/m³), g = 9.80665 m/s², P1 = 200 kPa, v1 = 2 m/s, z1 = 2 m, z2 = 0 m, v2 = 5 m/s, and hL = 1 m. Solve for P2.</p>
            <p>Use: <em>P2 = P1 + 0.5ρ(v1² − v2²) + ρg(z1 − z2 − hL)</em></p>
            <p>Compute terms: 0.5·ρ·(v1² − v2²) = 0.5·1000·(4 − 25) = −10500 Pa; ρg(z1 − z2 − hL) = 1000·9.80665·1 = 9806.65 Pa.</p>
            <p>Therefore, P2 ≈ 199306.65 Pa (≈ 199.31 kPa).</p>
          </section>

              <summary>When is Bernoulli applicable?</summary>

            <details>
              <summary>How do I include frictional losses?</summary>

    function compute(inputs) {

      const safePressureHead = (value) => Number.isFinite(value) ? value / (rho * g) : 0;

          solvedBase = Math.sqrt(Math.max(0, rad));
          break;
        }
        case 'z1':
          solvedBase = energy2 - (actual.P1 + actual.v1);
          break;
        case 'z2':
          solvedBase = energy1 - (actual.P2 + actual.v2) - actual.hL;
          break;
        default:

      const finalHead2 = safePressureHead(base.P2) + safeVelocityHead(base.v2) + normalize(base.z2) + actual.hL;
      const deltaP = base.P1 - base.P2;
      return { base, target, solvedBase, head1: finalHead1, head2: finalHead2, deltaP, status };
    }
