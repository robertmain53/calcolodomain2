        <h2 class="text-2xl font-semibold">How to Use This Calculator</h2>
        <p>Insert the distance from the blast point (in meters) and the explosive charge weight (kilograms). Use the Calculate button to commit the values. Results never display NaN or Infinity because the system enforces required positive inputs.</p>
        <h3 class="text-xl font-semibold mt-1">Methodology</h3>
        <p>This tool follows the simplified vibration model referenced in ASCE 7-22. The nonlinear relationship between distance and charge weight is captured through the cube root of the explosive mass, keeping the constant <em>k</em> locked at 1.0 for clarity. The right-hand panel shows the rounded result and a profile for quick sensitivity checks.</p>
        <h3 class="text-xl font-semibold mt-6">Glossary</h3>
        <ul class="list-disc list-inside text-gray-700">
          <li><strong>Vibration Level:</strong> Peak particle velocity measured in millimeters per second (mm/s).</li>
          <li><strong>Distance (D):</strong> Linear distance from the blast focus to the monitoring point, measured in meters.</li>
          <li><strong>Charge Weight (W):</strong> Total explosive mass used in the shot, expressed in kilograms.</li>
        </ul>
        <h3 class="text-xl font-semibold mt-6">Example</h3>
        <p>With a charge weight of 50 kg and a stand-off distance of 100 meters, the calculator reports about 0.04 mm/s, matching the simple ratio <code>V = (W¹ᐟ³) / D</code>. That output aligns with the explanation in the formulas section.</p>
        <h3 class="text-xl font-semibold mt-6">Frequently Asked Questions (FAQ)</h3>
        <div class="faqs space-y-4 text-gray-700">
          <div>
            <h4 class="font-semibold">What is blasting vibration?</h4>
            <p>Blasting vibration refers to the ground movement that occurs when explosives transfer energy to the surrounding rock and soil.</p>
          </div>
          <div>
            <h4 class="font-semibold">How does distance affect vibration levels?</h4>
            <p>Vibration levels drop quickly as the distance increases because the energy dissipates over a larger area.</p>
          </div>
          <div>
            <h4 class="font-semibold">Why is it important to calculate vibration levels?</h4>
            <p>Controlling vibration helps keep nearby structures and sensitive equipment safe from damage.</p>
          </div>
          <div>
            <h4 class="font-semibold">Can vibration levels be controlled?</h4>
            <p>Yes, by reducing the charge weight, adjusting the blast pattern, or increasing the standoff distance.</p>

          <div>
            <h4 class="font-semibold">What are the units of measurement?</h4>
            <p>The calculator reports vibration as millimeters per second (mm/s), which is the standard for peak particle velocity.</p>

            function debounce(fn, delay) {
                let timer;
                return function() {
                    const args = arguments;
                    clearTimeout(timer);
                    timer = setTimeout(function() {
                        fn.apply(null, args);
                    }, delay);
                };
            }

            function parseInputs() {
                const parsed = {};
                Object.entries(inputElements).forEach(([key, el]) => {
                    const raw = el.value.trim();
                    const value = raw === '' ? null : Number(raw);
                    parsed[key] = Number.isFinite(value) ? value : null;
                });
                return parsed;
            }

            function validate(inputs) {
                const errors = [];
                if (inputs.distance == null || inputs.distance <= 0) {
                    errors.push('Distance must be a positive number.');
                }
                if (inputs.chargeWeight == null || inputs.chargeWeight <= 0) {
                    errors.push('Charge weight must be a positive number.');
                }
                return { ok: errors.length === 0, errors };
            }

            function buildProfile(inputs) {
                const variations = [
                    { label: 'Short distance', distance: inputs.distance * 0.8, charge: inputs.chargeWeight },
                    { label: 'Base scenario', distance: inputs.distance, charge: inputs.chargeWeight },
                    { label: 'Long distance', distance: inputs.distance * 1.2, charge: inputs.chargeWeight },
                    { label: 'Heavier charge', distance: inputs.distance, charge: inputs.chargeWeight * 1.15 },
                    { label: 'Lighter charge', distance: inputs.distance, charge: inputs.chargeWeight * 0.85 }
                ];
                return variations.map(variant => {
                    const distance = Math.max(variant.distance, 0.01);
                    const chargeWeight = Math.max(variant.charge, 0.01);
                    const vibration = kConstant * Math.pow(chargeWeight, 1 / 3) / distance;
                    return {
                        scenario: variant.label,
                        distance,
                        chargeWeight,
                        vibration
                    };
                });
            }

            function compute(inputs) {
                const vibrationLevel = kConstant * Math.pow(inputs.chargeWeight, 1 / 3) / inputs.distance;
                const profile = buildProfile(inputs);
                return { vibrationLevel, profile };
            }

            function format(outputs) {
                const formatted = {
                    vibrationLevel: Number.isFinite(outputs.vibrationLevel) ? outputs.vibrationLevel.toFixed(2) + ' mm/s' : '—',
                    scheduleRows: outputs.profile.map((step, index) => ({
                        index: index + 1,
                        scenario: step.scenario,
                        distance: step.distance.toFixed(1),
                        chargeWeight: step.chargeWeight.toFixed(1),
                        vibration: step.vibration.toFixed(2)
                    }))
                };
                return formatted;
            }

            function render(formatted, outputs, errors) {
                if (errors && errors.length) {
                    errorBox.textContent = errors.join(' ');
                    errorBox.style.display = 'block';
                    vibrationOutput.textContent = '—';
                    scheduleBody.innerHTML = '<tr><td colspan="5" class="text-center text-gray-500">Provide valid inputs to populate the vibration profile.</td></tr>';
                    downloadCsvBtn.disabled = true;
                    lastSchedule = [];
                    return;
                }
                errorBox.style.display = 'none';
                vibrationOutput.textContent = formatted.vibrationLevel;
                scheduleBody.innerHTML = '';
                formatted.scheduleRows.forEach(row => {
                    const tr = document.createElement('tr');
                    tr.innerHTML = '\n                        <td class="font-medium text-gray-600">' + row.index + '</td>\n                        <td class="text-gray-700">' + row.scenario + '</td>\n                        <td class="text-gray-700">' + row.distance + '</td>\n                        <td class="text-gray-700">' + row.chargeWeight + '</td>\n                        <td class="font-semibold text-blue-700">' + row.vibration + '</td>\n                    ';
                    scheduleBody.appendChild(tr);
                });
                lastSchedule = outputs.profile;
                downloadCsvBtn.disabled = false;
            }

            function update() {
                const parsed = parseInputs();
                const validation = validate(parsed);
                if (!validation.ok) {
                    render(null, null, validation.errors);
                    return;
                }
                const outputs = compute(parsed);
                const formatted = format(outputs);
                render(formatted, outputs);
            }

            const debouncedUpdate = debounce(update, 100);

            calcBtn.addEventListener('click', update);
            resetBtn.addEventListener('click', () => {
                Object.values(inputElements).forEach(el => {
                    if (el.dataset.default != null) {
                        el.value = el.dataset.default;
                    }
                });
                update();
            });

            Object.values(inputElements).forEach(el => {
                el.addEventListener('input', debouncedUpdate);
            });

            toggleScheduleBtn.addEventListener('click', () => {
                scheduleVisible = !scheduleVisible;
                scheduleWrap.style.display = scheduleVisible ? 'block' : 'none';
                toggleScheduleBtn.textContent = scheduleVisible ? 'Hide Vibration Profile' : 'View Vibration Profile';
                toggleScheduleBtn.setAttribute('aria-expanded', String(scheduleVisible));
            });

            downloadCsvBtn.addEventListener('click', () => {
                if (!lastSchedule.length) {
                    return;
                }
                const csvRows = ['Scenario,Distance (m),Charge Weight (kg),Vibration (mm/s)'];
                lastSchedule.forEach(step => {
                    csvRows.push(step.scenario + ',' + step.distance.toFixed(3) + ',' + step.chargeWeight.toFixed(3) + ',' + step.vibration.toFixed(3));
                });
                const blob = new Blob([csvRows.join('\n')], { type: 'text/csv' });
                const url = URL.createObjectURL(blob);
                const anchor = document.createElement('a');
                anchor.href = url;
                anchor.download = 'blasting-vibration-profile.csv';
                document.body.appendChild(anchor);
                anchor.click();
                document.body.removeChild(anchor);
                URL.revokeObjectURL(url);
            });

            update();
        })();
    </script>
</body></html>
