      <p>Enter the total at-bats and hits for the period you want to evaluate. Click Calculate (or let the inputs update) to see your updated batting average immediately. The Reset button returns the defaults and recomputes the value.</p>
      <h2 class="text-2xl font-semibold">Methodology</h2>
      <div class="bg-gray-50 border border-gray-200 rounded-lg p-4 text-sm text-gray-700">
        <p>We use the standard baseball batting average definition: hits divided by at-bats. The calculator enforces valid ranges (at-bats > 0, hits between 0 and at-bats) so the output remains between 0.000 and 1.000, mirroring league statistics methodology.</p>
        <p class="mt-2">Factors like walks, hit-by-pitch, and sacrifices are excluded because they are not counted as official at-bats in the rules book.</p>
      </div>
      <h2 class="text-2xl font-semibold">Glossary of Terms</h2>
      <ul class="text-gray-700">
        <li><strong>At Bats:</strong> The number of times a player has batted, excluding walks, hits by pitch, sacrifices, catcher interference, and obstruction.</li>
        <li><strong>Hits:</strong> Instances when a player safely reaches first base (or beyond) after hitting the ball, without the aid of an error or fielder's choice.</li>
        <li><strong>Batting Average:</strong> Offensive performance measure equal to hits divided by at-bats, presented as a decimal between 0.000 and 1.000.</li>
      </ul>

        <div>
          <h3 class="font-semibold">Does a batting average include walks?</h3>
          <p>No, walks, hit-by-pitch, and sacrifices do not count as at-bats, so they are excluded from the batting average calculation.</p>

        <div>
          <h3 class="font-semibold">Can a batting average be negative?</h3>

          atBats: Number.isFinite(inputs.atBats) ? inputs.atBats.toLocaleString('en-US', { maximumFractionDigits: 0 }) : '0'
        };
      }
      function render(formatted, errors) {
        if (errors && errors.length) {
          els.errorBox.textContent = errors.join(' ');
          els.errorBox.style.display = 'block';
          els.battingAverage.textContent = '0.000';
          els.hitsValue.textContent = '0';
          els.atBatsValue.textContent = '0';
          return;
        }
        els.errorBox.style.display = 'none';
        els.errorBox.textContent = '';
        if (!formatted) return;
        els.battingAverage.textContent = formatted.battingAverage;
        els.hitsValue.textContent = formatted.hits;
        els.atBatsValue.textContent = formatted.atBats;
      }
      function update() {
        const inputs = parseInputs();
        const validation = validate(inputs);
        if (!validation.ok) {
          render(null, validation.errors);
          return;
        }
        const outputs = compute(inputs);
        if (!outputs) {
          render(null, ['Unable to calculate batting average with the provided values.']);
          return;
        }
        const formatted = format(outputs, inputs);
        render(formatted, []);
      }
      const debouncedUpdate = debounce(update, 100);
      els.calcBtn.addEventListener('click', update);
      els.resetBtn.addEventListener('click', () => {
        els.atBats.value = defaults.atBats;
        els.hits.value = defaults.hits;
        update();
      });
      [els.atBats, els.hits].forEach((input) => {
        input.addEventListener('input', debouncedUpdate);
        input.addEventListener('change', debouncedUpdate);
      });
      update();
    })();
  </script>
