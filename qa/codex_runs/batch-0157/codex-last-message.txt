        <h2 class="text-2xl font-semibold text-gray-900">How to Use This Calculator</h2>
        <p class="mt-3 text-gray-700">Enter the count of independent trials, the probability that each trial succeeds, and the target number of successes. Click "Calculate" to refresh the result or change any input and let the form recompute automatically after a short debounce.</p>
        <h3 class="text-xl font-semibold mt-6">Methodology</h3>
        <p class="mt-2 text-gray-700">The calculator applies the binomial probability mass function directly: it multiplies the binomial coefficient by powers of the success and failure probabilities to compute the likelihood of exactly <em>k</em> successes.</p>
        <ul class="list-disc list-inside mt-3 text-gray-700 space-y-2">
          <li>The binomial coefficient counts how many sequences produce exactly <em>k</em> successes.</li>
          <li>Each success contributes a factor of <em>p</em>, each failure a factor of (1 - <em>p</em>).</li>
          <li>Inputs are validated to keep probabilities between 0 and 1, and to ensure successes do not exceed the trial count.</li>
        </ul>

          <h3 class="text-xl font-semibold">Full original guide (expanded)</h3>
          <p>This calculator is designed for statisticians and data analysts to compute the probability of obtaining a given number of successes in a fixed number of trials with a constant probability of success.</p>
          <section aria-labelledby="original-methodology">
            <h4 id="original-methodology" class="text-lg font-semibold mt-4">Data Source and Methodology</h4>
            <p>All calculations are based strictly on the formulas and data provided by authoritative statistical sources.</p>
            <h4 class="text-lg font-semibold mt-4">The Formula Explained</h4>
            <p>The probability of getting exactly k successes in n trials is given by:</p>
            <div class="formula-box">
              \( P(X = k) = \binom{n}{k} p^k (1-p)^{n-k} \)
            </div>
            <h4 class="text-lg font-semibold mt-4">Glossary of Variables</h4>
            <ul class="list-disc list-inside text-gray-700 space-y-1">
              <li><strong>n:</strong> Number of trials.</li>
              <li><strong>p:</strong> Probability of success on an individual trial.</li>
              <li><strong>k:</strong> Number of successes.</li>
              <li><strong>\( \binom{n}{k} \):</strong> Binomial coefficient, representing the number of ways to choose k successes from n trials.</li>
            </ul>
            <h4 class="text-lg font-semibold mt-4">How It Works: A Step-by-Step Example</h4>
            <p>Suppose you flip a biased coin 10 times, where the probability of heads (success) is 0.5. What is the probability of getting exactly 5 heads?</p>
            <p>Using the formula: \( P(X = 5) = \binom{10}{5} (0.5)^5 (1-0.5)^{10-5} \). Calculate each component to find the probability.</p>
            <h4 class="text-lg font-semibold mt-4">Frequently Asked Questions (FAQ)</h4>
            <h5 class="text-base font-semibold mt-3">What is the Binomial Distribution?</h5>
            <p>The Binomial Distribution represents the number of successes in a sequence of n independent experiments, each asking a yes/no question, and each with its own boolean-valued outcome.</p>
            <h5 class="text-base font-semibold mt-3">How do I use this calculator?</h5>
            <p>Enter the number of trials, the probability of success, and the number of successes, then click 'Calculate' to see the probability.</p>
            <h5 class="text-base font-semibold mt-3">What is the difference between Binomial and Normal distribution?</h5>
            <p>The Binomial Distribution is discrete, while the Normal Distribution is continuous. They are related through the Central Limit Theorem.</p>
            <h5 class="text-base font-semibold mt-3">Can the probability be greater than 1?</h5>
            <p>No, probabilities range from 0 to 1.</p>
            <h5 class="text-base font-semibold mt-3">What applications use the Binomial Distribution?</h5>
            <p>Common applications include quality control, finance, and health sciences where binary outcomes are analyzed.</p>

          }
          if (!Number.isFinite(inputs.successes) || inputs.successes < 0 || !Number.isInteger(inputs.successes)) {
            errors.push('Number of successes must be an integer 0 or greater.');
          }
          if (Number.isFinite(inputs.trials) && Number.isFinite(inputs.successes) && inputs.successes > inputs.trials) {
            errors.push('Successes cannot exceed the number of trials.');
          }
          return { ok: errors.length === 0, errors };
        }

        function binomialCoefficient(n, k) {
          if (k < 0 || k > n) return 0;
          k = Math.min(k, n - k);
          let coeff = 1;
          for (let i = 1; i <= k; i++) {
            coeff = coeff * (n - k + i) / i;
          }
          return coeff;
        }

        function compute(inputs) {
          const { trials: n, successProb: p, successes: k } = inputs;
          const coefficient = binomialCoefficient(n, k);
          if (!Number.isFinite(coefficient) || coefficient < 0) {
            return { ok: false, errors: ['Unable to compute the binomial coefficient.'] };
          }
          let probability = coefficient * Math.pow(p, k) * Math.pow(1 - p, n - k);
          if (!Number.isFinite(probability) || probability < 0) {
            probability = 0;
          }
          return { ok: true, probability };
        }

        function formatProbability(value) {
          return Number.isFinite(value) ? value.toFixed(4) : '0.0000';
        }

        function format(outputs, inputs) {
          return {
            probability: formatProbability(outputs.probability),
            trialsLabel: Number.isFinite(inputs.trials) ? inputs.trials : '—',
            successesLabel: Number.isFinite(inputs.successes) ? inputs.successes : '—'
          };
        }

        function render(formatted, errors) {
          const probabilityEl = document.getElementById('probabilityValue');
          const trialsEl = document.getElementById('trialsLabel');
          const successesEl = document.getElementById('successesLabel');
          const errorBox = document.getElementById('errorBox');

          if (errors && errors.length > 0) {
            errorBox.style.display = 'block';
            errorBox.textContent = errors.join(' ');
            probabilityEl.textContent = '—';
            trialsEl.textContent = '—';
            successesEl.textContent = '—';
            return;
          }

          errorBox.style.display = 'none';
          errorBox.textContent = '';
          probabilityEl.textContent = formatted.probability;
          trialsEl.textContent = formatted.trialsLabel;
          successesEl.textContent = formatted.successesLabel;
        }

        function update() {
          const inputs = parseInputs();
          const validation = validate(inputs);
          if (!validation.ok) {
            render(null, validation.errors);
            return;
          }

          const result = compute(inputs);
          if (!result.ok) {
            render(null, result.errors);
            return;
          }

          const formatted = format(result, inputs);
          render(formatted, []);
        }

        const debouncedUpdate = debounce(update, 100);

        function resetForm() {
          document.getElementById('numTrials').value = DEFAULTS.trials;
          document.getElementById('probSuccess').value = DEFAULTS.successProb;
          document.getElementById('numSuccesses').value = DEFAULTS.successes;
          update();
        }

        document.getElementById('calcBtn').addEventListener('click', update);
        document.getElementById('resetBtn').addEventListener('click', resetForm);

        document.querySelectorAll('#inputsCard input').forEach((input) => {
          input.addEventListener('input', debouncedUpdate);
          input.addEventListener('change', debouncedUpdate);
        });

        resetForm();
      })();
    </script>
