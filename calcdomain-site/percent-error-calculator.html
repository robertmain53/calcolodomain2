The Formula Explained

Difference (error): \[ e = x_o - x_t \]

Absolute percent error: \[ \%\mathrm{Error}_{abs} = \frac{|x_o - x_t|}{|x_t|}\times 100\% \quad (x_t \ne 0) \]

Signed percent error: \[ \%\mathrm{Error}_{signed} = \frac{x_o - x_t}{x_t}\times 100\% \quad (x_t \ne 0) \]

How It Works: A Step-by-Step Example

  1. Inputs. \(x_o = 9.8\), \(x_t = 10.0\).
  2. Difference. \(e = 9.8 - 10.0 = -0.2\).
  3. Absolute % error. \(\frac{|{-0.2}|}{10.0}\times 100\% = 2\%\).
  4. Signed % error. \(\frac{-0.2}{10.0}\times 100\% = -2\%\).

FAQ

What is percent error?

It expresses the difference between a measured (observed) value and a true (accepted) value as a percentage of the true value.

Which convention should I use?

For quality control and reporting an overall deviation, absolute percent error is common. When direction matters (over- vs under-estimate), use the signed convention.

What if the true value is zero?

Percent error is undefined because it divides by \(x_t\). Use absolute error \(e\) or a different metric.

How many decimals should I report?

Match the measurement context or standard in your field. Use the “Display decimals” control for presentation.

Can I process many values at once?

Yes. Switch to Batch mode and paste/edit rows; results compute per row in real time.

Is percent error the same as relative error?

Relative error is \(|x_o-x_t|/|x_t|\); percent error is that value × 100%.
