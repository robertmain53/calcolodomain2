Dodecagon formulas used

n = 12 (regular dodecagon)

Perimeter: \(P = 12a\)

Area (exact): \(A = 3(2 + \sqrt{3}) \, a^2\)

General regular polygon form: \(A = \frac{n a^2}{4 \tan(\pi/n)}\) → for \(n=12\): \(A = \frac{12 a^2}{4 \tan(\pi/12)} = \frac{3 a^2}{\tan(15^\circ)} = 3(2+\sqrt{3})a^2\)

Apothem: \(r = \frac{a}{2 \tan(\pi/12)}\)

Circumradius: \(R = \frac{a}{2 \sin(\pi/12)}\)

Diameter (vertex-to-vertex): \(D = 2R\)

Dodecagonal prism volume: \(V = A \cdot h\)

FAQs

Is this for regular or irregular dodecagons?

This tool is for regular dodecagons (all sides equal). For arbitrary 12-gons with different vertices, use the Irregular Polygon Calculator.

Can I use flat-to-flat instead?

For a dodecagon, “flat-to-flat” isn’t as standard as for hexagons. Use apothem (distance from center to a side) — it’s effectively half of flat-to-flat.

What’s special about 12 sides?

12 = 3 × 4, so a dodecagon can tile nicely and it admits exact radical formulas, which is why you see the expression 3(2+√3)a².
