What are hyperbolic functions?

Hyperbolic functions are cousins of the circular trigonometric functions, but instead of being based on the unit circle, they are defined using exponential functions and related to the geometry of hyperbolas. For a real variable \(x\), the basic hyperbolic functions are:

Definitions via exponentials

  • \(\displaystyle \sinh x = \frac{e^x - e^{-x}}{2}\)
  • \(\displaystyle \cosh x = \frac{e^x + e^{-x}}{2}\)
  • \(\displaystyle \tanh x = \frac{\sinh x}{\cosh x}\)

Reciprocal functions are defined by \(\displaystyle \operatorname{sech} x = \frac{1}{\cosh x}\), \(\displaystyle \operatorname{csch} x = \frac{1}{\sinh x}\), \(\displaystyle \coth x = \frac{\cosh x}{\sinh x}\), whenever the denominators are nonzero.

The graphs of \(\sinh x\), \(\cosh x\), and \(\tanh x\) resemble their trigonometric counterparts in some ways, but they are not periodic and grow roughly exponentially for large \(|x|\).

Core hyperbolic identities

Pythagorean-type identity

  • \(\cosh^2 x - \sinh^2 x = 1\)

Compare with the circular identity \(\cos^2 x + \sin^2 x = 1\): the sign in front of \(\sinh^2 x\) is negative instead of positive.

Addition formulas

  • \(\sinh(x + y) = \sinh x \cosh y + \cosh x \sinh y\)
  • \(\cosh(x + y) = \cosh x \cosh y + \sinh x \sinh y\)
  • \(\tanh(x + y) = \dfrac{\tanh x + \tanh y}{1 + \tanh x \tanh y}\), whenever \(1 + \tanh x \tanh y \neq 0\)

Double-angle formulas

  • \(\sinh(2x) = 2\sinh x \cosh x\)
  • \(\cosh(2x) = \cosh^2 x + \sinh^2 x\)
  • \(\tanh(2x) = \dfrac{2\tanh x}{1 + \tanh^2 x}\), whenever \(1 + \tanh^2 x \neq 0\)

Inverse hyperbolic functions

Each basic hyperbolic function has an inverse, defined on appropriate domains. Useful closed forms for real \(x\) are:

  • \(\displaystyle \operatorname{arsinh} x = \ln\bigl(x + \sqrt{x^2 + 1}\bigr)\), all real \(x\)
  • \(\displaystyle \operatorname{arcosh} x = \ln\bigl(x + \sqrt{x^2 - 1}\bigr)\), \(x \ge 1\)
  • \(\displaystyle \operatorname{artanh} x = \tfrac{1}{2}\ln\left(\frac{1 + x}{1 - x}\right)\), \(|x| < 1\)
  • \(\displaystyle \operatorname{arcoth} x = \tfrac{1}{2}\ln\left(\frac{x + 1}{x - 1}\right)\), \(|x| > 1\)
  • \(\displaystyle \operatorname{arsech} x = \operatorname{arcosh}\left(\frac{1}{x}\right)\), \(0 < x \le 1\)
  • \(\displaystyle \operatorname{arcsch} x = \operatorname{arsinh}\left(\frac{1}{x}\right)\), \(x \neq 0\)

Connections to trigonometry and complex analysis

Hyperbolic functions mirror many trigonometric identities, with sign changes reflecting the geometry of hyperbolas instead of circles. For example:

  • Trigonometric: \(\cos^2 x + \sin^2 x = 1\)
  • Hyperbolic: \(\cosh^2 x - \sinh^2 x = 1\)

Using complex numbers, the two families are tied together through Euler’s formula \(e^{ix} = \cos x + i\sin x\). For instance:

  • \(\sin(ix) = i \sinh x\)
  • \(\cos(ix) = \cosh x\)

In practice, hyperbolic functions are everywhere: in the shape of a hanging cable (the catenary \(y = a\cosh(x/a)\)), in solutions to differential equations, and in the rapidity formalism of special relativity.
