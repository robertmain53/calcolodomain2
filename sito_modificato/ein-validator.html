Formula (extracted text)
Let $E_s$ be the input string (EIN). Let $E_c$ be the "cleaned" string, $E_s$, with all non-numeric characters removed. Let $P$ be the prefix of $E_c$, $P = \text{substring}(E_c, 0, 2)$. Let $S$ be the set of valid IRS prefixes: $S = \{10, 12, 13, 14, 16, 20, 21, \dots, 99\}$. $\text{ValidFormat} = (\text{length}(E_c) = 9) \land (P \in S)$
