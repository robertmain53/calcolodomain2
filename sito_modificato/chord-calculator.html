Chord Length Calculator

Compute the length of a chord in a circle using the central angle, the sagitta (height), or the arc length. Designed for students, engineers, and drafters who need precise, fast results with clear formulas and a live diagram.

The Formula Explained

For radius \(R\) and central angle \(\theta\) (radians):
\[ c = 2R\sin\!\left(\frac{\theta}{2}\right) \]
Converting degrees to radians: \(\theta_{\text{rad}} = \theta_{\deg}\cdot \frac{\pi}{180}\).

From sagitta \(s\) (minor segment):
\[ c = 2\sqrt{2Rs - s^2}, \quad 0 < s \le R \]

From arc length \(L\): since \(L = R\theta\), then \(\theta = \frac{L}{R}\) and
\[ c = 2R\sin\!\left(\frac{L}{2R}\right) \]

Glossary of Variables

  • R — circle radius (any length unit; all outputs keep the same unit).
  • θ — central angle subtending the chord (degrees shown; converted to radians internally).
  • c — chord length.
  • s — sagitta (height from chord to arc, minor segment).
  • L — arc length along the circle between chord endpoints.

How It Works: A Step-by-Step Example

Given: \(R = 10\), \(\theta = 60^\circ\).

  1. Convert \(60^\circ\) to radians: \(\theta = 60\cdot\pi/180 = \pi/3\).
  2. Apply \(c = 2R\sin(\theta/2) = 2(10)\sin(\pi/6) = 20 \cdot 0.5 = 10\).
  3. Arc length: \(L = R\theta = 10\cdot \pi/3 \approx 10.472\).
  4. Sagitta: \(s = R - \sqrt{R^2 - (c/2)^2} = 10 - \sqrt{100 - 25} \approx 10 - 8.660 = 1.340\).

Result: \(c=10\), \(L\approx 10.472\), \(s\approx 1.340\).

Frequently Asked Questions

What is a chord?

A chord is a straight line segment whose endpoints lie on the circle.

Is the chord unique for a given angle?

Yes, for a fixed radius the chord length depends only on the subtended central angle.

Can a chord be longer than the diameter?

No. The diameter is the longest chord; any other chord is shorter.

What’s the valid range for sagitta?

For the minor segment convention, \(0 < s \le R\). At \(s=R\), the chord is a diameter.

How do units work?

Use any length unit for inputs; all derived results use the same unit.

Angle in radians vs degrees?

The calculator accepts degrees for convenience and converts to radians internally for trigonometric formulas.
