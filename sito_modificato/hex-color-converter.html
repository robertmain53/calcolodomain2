How this hex color converter works

This tool is bidirectional: you can start from HEX, RGB, HSL or the color picker. As soon as one value changes, all other formats are recalculated and kept in sync.

It supports:

  • 3- and 6-digit hex: #F53, #FF5733
  • 4- and 8-digit hex with alpha: #F53C, #FF5733CC
  • rgb() and rgba() functions
  • hsl() and hsla() functions
  • Derived formats: CMYK, CIELAB, LCH and CIEXYZ

1. HEX to RGB formula

For a 6-digit hex color #RRGGBB:

RR, GG, BB = two-digit hex values (00–FF)

R = decimal(RR)
G = decimal(GG)
B = decimal(BB)

Example for #FF5733:

  • FF in hex = 255 in decimal → red
  • 57 in hex = 87 in decimal → green
  • 33 in hex = 51 in decimal → blue

So the RGB value is rgb(255, 87, 51).

2. RGB to HEX formula

For an RGB color rgb(R, G, B) with each channel between 0 and 255: 

RR = hex(R).padStart(2, '0')
GG = hex(G).padStart(2, '0')
BB = hex(B).padStart(2, '0')

HEX = '#' + RR + GG + BB

Example for rgb(255, 87, 51):

  • 255 → FF
  • 87 → 57
  • 51 → 33

So the hex code is #FF5733.

3. RGB to HSL (Hue, Saturation, Lightness)

First normalize RGB to the range 0–1:

r = R / 255
g = G / 255
b = B / 255

max = max(r, g, b)
min = min(r, g, b)
L = (max + min) / 2

Then compute saturation S and hue H: 

if max == min:
    H = 0
    S = 0
else:
    d = max - min
    S = d / (1 - |2L - 1|)

    if max == r:
        H = 60 * (((g - b) / d) mod 6)
    else if max == g:
        H = 60 * (((b - r) / d) + 2)
    else:
        H = 60 * (((r - g) / d) + 4)

Finally, express H in degrees (0–360), and S, L as percentages (0–100%).

4. RGB to CMYK approximation

CMYK is device-dependent, but a common approximation from RGB is: 

R', G', B' = R/255, G/255, B/255
K = 1 - max(R', G', B')

if K == 0:
    C = 0
    M = 0
    Y = 0
else:
    C = (1 - R' - K) / (1 - K)
    M = (1 - G' - K) / (1 - K)
    Y = (1 - B' - K) / (1 - K)

Multiply C, M, Y, K by 100 to get percentages.

5. LAB, LCH and XYZ

LAB, LCH and XYZ are perceptual color spaces used in color science and advanced design workflows. This converter uses the standard sRGB → XYZ → LAB pipeline with D65 white point and derives LCH from LAB.

These formats are useful when you want uniform lightness steps, perceptual contrast, or to build OKLCH-like palettes.

Hex Color Converter FAQ

What is a hex color code?

A hex color code is a 3-, 4-, 6- or 8-digit hexadecimal value that encodes the red, green, blue and optional alpha (transparency) channels of a color. For example, #FF5733 is an opaque orange, while #FF573380 is the same color at 50% opacity.

Can I convert from RGB or HSL back to hex?

Yes. This tool is fully bidirectional. If you edit the RGB or HSL fields, the hex value is recalculated instantly. You can use whichever format is most comfortable and copy the others as needed.

Why do my CMYK values look different from my printer software?

CMYK depends on the specific printer, ink and paper profile. The values shown here are a standard mathematical approximation from sRGB and are suitable for quick estimates, but professional print workflows should rely on ICC profiles and your print provider’s tools.

What are LAB and LCH used for?

LAB and LCH are perceptually uniform color spaces. Equal steps in lightness or chroma correspond more closely to equal perceived changes. They are useful for building accessible palettes, smooth gradients and data visualizations where consistent contrast matters.

Does this converter support transparency (alpha)?

Yes. You can use 4- or 8-digit hex codes (e.g. #RGBA or #RRGGBBAA) or RGBA/HSLA functions. The alpha channel is shown as a percentage slider and is included in the RGBA and HSLA outputs.
