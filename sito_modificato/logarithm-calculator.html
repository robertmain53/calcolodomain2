The formula explained

\[ y = \log_b(x) \iff b^{y} = x,\quad b>0,\ b\ne 1,\ x>0 \] \[ \log_b(x) = \frac{\ln x}{\ln b} = \frac{\log_{10} x}{\log_{10} b} \] \[ \text{Antilog:}\; x = b^{y},\ \text{with special cases } e^{y}\ (\ln)\ \text{and } 10^{y}\ (\log_{10}). \]

How it works: a step‑by‑step example

Example: Compute \(\log_2(8)\)

  1. Inputs: x = 8, b = 2, mode = Logarithm.
  2. Apply change of base: \(\log_2(8) = \ln(8)/\ln(2)\).
  3. Since \(8 = 2^3\), we expect \(\log_2(8) = 3\).
  4. The calculator returns 3.0000 (with 4 decimals).

FAQ

Is base e the same as ln?

Yes. ln(x) means log base e of x, where e ≈ 2.718281828…

Can I convert a known log value from base 10 to base 2?

Yes. If you know v = log 10 (x), then log 2 (x) = v·ln(10)/ln(2).

Why are inputs like x ≤ 0 rejected?

Logarithms are defined only for positive arguments. This is a core property of the function's domain.

What about precision and rounding?

Internal math uses IEEE‑754 double precision. We round only at display according to your selected decimal places.

Does the tool support scientific notation?

Yes. You can enter values like 1e-6 or 3.5e8 in the number fields.

How can I get the antilog quickly?

Switch to Antilog mode and enter base b and exponent y; the tool computes x = b^y.
