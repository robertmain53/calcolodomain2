Data Source & Methodology

Authoritative source: OpenStax Prealgebra 2e, Section 5 “Fractions” (2015; updated 2022). All computations are exact on rational numbers using integer arithmetic (no floating-point rounding during fraction operations). Results are reduced using the greatest common divisor (GCD). :contentReference[oaicite:3]{index=3}

The Formula Explained

Add/Subtract

\[ \frac{a}{b} \pm \frac{c}{d} = \frac{ad \pm bc}{\,bd\,} \]

Multiply

\[ \frac{a}{b} \times \frac{c}{d} = \frac{ac}{\,bd\,} \]

Divide (with \(c \neq 0\))

\[ \frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c} = \frac{ad}{\,bc\,} \]

Simplify with GCD

\[ g=\gcd(n,d),\quad \frac{n}{d}=\frac{n/g}{\,d/g\,} \]

Mixed Number from Improper

\[ n = qd + r,\ 0 \le r < d \;\Rightarrow\; \frac{n}{d} = q\ \frac{r}{d} \]

How It Works: A Step-by-Step Example

Compute \(1\ \tfrac{1}{2} + \tfrac{3}{4}\).

  1. Normalize inputs to improper fractions: \(1\ \tfrac{1}{2} = \tfrac{3}{2}\); \(\tfrac{3}{4}\) is already proper.
  2. Apply addition: \(\tfrac{3}{2} + \tfrac{3}{4} = \tfrac{3\cdot4 + 3\cdot2}{2\cdot4} = \tfrac{18}{8}\).
  3. Simplify using \(\gcd(18,8)=2\) → \(\tfrac{9}{4}\).
  4. Convert to mixed number: \(\tfrac{9}{4} = 2\ \tfrac{1}{4}\). Decimal: 2.25.

Frequently Asked Questions (FAQ)

What formats can I enter?

Integers (e.g., -3), fractions (e.g., 7/8), mixed numbers (e.g., 1 3/4), or decimals (e.g., 0.125). Use a space between whole and fractional parts. :contentReference[oaicite:4]{index=4}

How do you avoid rounding errors?

All fraction arithmetic is done using integer numerators and denominators (BigInt). Decimal inputs are converted to exact rationals before computation; simplification uses GCD. :contentReference[oaicite:5]{index=5}

Why is division by zero blocked?

Division by zero is undefined. If the second operand equals 0 and you choose Divide, the tool shows a clear error and prevents calculation. :contentReference[oaicite:6]{index=6}

Can I disable simplification?

Yes. Toggle “Auto-simplify” to view the raw unsimplified fraction (useful for checking intermediate work). :contentReference[oaicite:7]{index=7}

Does it support negative mixed numbers?

Yes. Enter them as “-3 1/2”. The negative sign applies to the entire mixed number. :contentReference[oaicite:8]{index=8}
