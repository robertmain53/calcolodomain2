The Formula Explained

Skins are calculated by comparing player scores on each hole. The player with the lowest score wins the skin for that hole.

Glossary of Terms

  • Player Count: Total number of players in the game.
  • Holes Count: Total number of holes played.
  • Skin: A point awarded to the player with the lowest score on a hole.

Frequently Asked Questions (FAQ)

What is a Skins Game?

A skins game is a type of golf match where each hole has a set value or "skin". The player who wins a hole outright wins the skin for that hole.

How are ties handled in skins?

If two or more players tie for the lowest score, the skin is carried over to the next hole.

Can skins be shared?

No, skins are awarded only to a player who wins a hole outright.

What happens if no one wins a skin?

If no player wins a hole outright, the skin is added to the next hole.

How do I input scores?

Enter the number of players and holes, then input individual scores for each hole through the interactive interface.
