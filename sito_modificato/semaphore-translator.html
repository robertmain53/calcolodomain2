Semaphore Flag Translator

Turn any word, call sign, or short instruction into International Code of Signals semaphore movements. Each output shows the exact right- and left-hand positions so you can practise or brief a team.

Letters A–Z are supported. Spaces insert a rest (both flags down). Numerals require the semaphore numeral indicator before the letter equivalents.

Data Source & Methodology

Positions follow the International Code of Signals (ICS), 2005 two-flag semaphore standard: eight clock-face positions numbered clockwise. Each card lists the descriptive direction and clock reference so trainees can visualise the pose without a diagram.

  • International Maritime Organization — ICS
  • USCG Navigation Center: Semaphore basics

Clock Positions

The eight semaphore positions correspond to clock-face angles:

  • 1: Up (12 o’clock)
  • 2: Up-right (1:30)
  • 3: Right (3 o’clock)
  • 4: Down-right (4:30)
  • 5: Down (6 o’clock)
  • 6: Down-left (7:30)
  • 7: Left (9 o’clock)
  • 8: Up-left (10:30)

Sending Numerals

Results flag digits because semaphore requires a numeral indicator (sometimes called the numeric flag). After showing the indicator, transmit letters A–J to convey 1–9 and 0 respectively. Return to letters by sending the alphabetic indicator (letters, signals, or cancel).

Practise Example

Input: ALPHA

  1. A → Right Up (12 o’clock), Left Up-right (1:30)
  2. L → Right Up-right (1:30), Left Up-left (10:30)
  3. P → Right Right (3 o’clock), Left Left (9 o’clock)
  4. H → Right Up-right (1:30), Left Down-right (4:30)
  5. A → repeat first position. Rest both flags Down between words.
