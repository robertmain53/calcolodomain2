    How to Use This Calculator

    Enter the overall room dimensions and each clearance measurement in inches. The calculator evaluates every value against the NKBA and ADA guidance embedded in the rule set, so you can see which clearances pass or fail before you finalize the layout.

    Methodology

    Each clearance is checked against the authoritative minimum listed in the NKBA guidelines. The engine flags values below the suggested thresholds and reports how far you are from compliance, while room dimensions provide context but are not part of the pass/fail summary.

    • Only measurements provided in the form are evaluated; empty fields are ignored until a value is entered.
    • Results show pass/fail status with rounded feedback so you can confidently explain how much extra clearance you have or need.
    • Room dimensions are preserved for reference and to help you think through circulation and fixture placement.
    Results are estimates and intended for planning. Always verify final layouts with local building codes and licensed professionals.

    Why Use This Calculator?

    • Avoid Costly Mistakes: Verify your plan before you build or buy fixtures.
    • Ensure Comfort: Check for adequate space to move and use each fixture without feeling cramped.
    • Plan for Accessibility: Use ADA-recommended values (like 32" doorways) to create a space that's usable for everyone.
    • Instant Feedback: Get immediate pass/fail results as you type, based on industry-standard guidelines.
