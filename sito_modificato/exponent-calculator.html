The Laws of Exponents

Exponentiation is governed by a few fundamental rules, or laws, that simplify multiplication and division of powers.

Law Formula Example
Product Rule $b^m \cdot b^n = b^{m+n}$ $2^3 \cdot 2^2 = 2^5 = 32$
Quotient Rule $$\frac{b^m}{b^n} = b^{m-n}$$ $$\frac{3^5}{3^2} = 3^3 = 27$$
Power Rule $(b^m)^n = b^{m \cdot n}$ $(4^2)^3 = 4^6 = 4096$
Negative Exponent Rule $$b^{-n} = \frac{1}{b^n}$$ $$5^{-2} = \frac{1}{5^2} = \frac{1}{25}$$
Zero Exponent Rule $b^0 = 1, \text{ for } b \neq 0$ $78^0 = 1$

Fractional and Decimal Exponents

When the exponent is a fraction or decimal, it relates directly to taking a root. A fractional exponent $n = \frac{p}{q}$ is handled by the Root Law:

$$b^{p/q} = \sqrt[q]{b^p} = (\sqrt[q]{b})^p$$

The denominator, $q$, determines the root (e.g., square root, cube root), and the numerator, $p$, determines the power.

Example: $8^{2/3}$ is the cube root of $8$ (which is $2$), raised to the power of $2$ (which is $4$).

Frequently Asked Questions (FAQ)

What is the rule for a negative exponent?

The rule for a negative exponent is: $b^{-n} = \frac{1}{b^n}$. A negative exponent simply means you take the reciprocal of the base raised to the positive exponent. Example: $2^{-3} = \frac{1}{2^3} = \frac{1}{8}$.

What is the rule for a fractional exponent?

A fractional exponent, like $b^{p/q}$, represents a root. The denominator ($q$) is the type of root (e.g., square root if $q=2$), and the numerator ($p$) is the power. The rule is: $b^{p/q} = \sqrt[q]{b^p}$. Example: $25^{1/2} = \sqrt{25} = 5$.

What is any number raised to the power of zero?

Any non-zero number raised to the power of zero is always 1. For example, $10^0 = 1$ and $(-5)^0 = 1$. The expression $0^0$ is generally considered undefined in algebra, although it may be treated as 1 in some contexts (like calculus).

How do you handle negative bases?

If the base is negative:

  • If the exponent is an **even** integer, the result is positive (e.g., $(-2)^4 = 16$).
  • If the exponent is an **odd** integer, the result is negative (e.g., $(-2)^3 = -8$).
  • If the exponent is fractional or decimal, the result may involve complex numbers (e.g., $\sqrt{-4}$).
