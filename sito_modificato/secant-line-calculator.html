The Formula Explained

For a function \(f(x)\) and two points \(x_1\) and \(x_2\) (with \(x_2 \neq x_1\)), the secant slope is:
\[ m \;=\; \frac{f(x_2) - f(x_1)}{x_2 - x_1} \] The slope–intercept equation of the secant line is:
\[ y \;=\; m\,x + b,\quad \text{where } b \;=\; f(x_1) - m\,x_1. \] Using a base point \(x_0\) and step \(h\) (so \(x_2 = x_0 + h\)):
\[ m \;=\; \frac{f(x_0+h) - f(x_0)}{h},\quad h \neq 0. \] This \(m\) is the average rate of change over the interval. As \(h \to 0\), the secant approaches the tangent.

Glossary of Variables

  • \(f(x)\): The input function in variable x.
  • \(x_1, x_2\): Two x-values where the secant intersects the graph.
  • \(x_0, h\): A base point and a small step (\(x_2 = x_0+h\)).
  • \(m\): Secant slope \(= \dfrac{f(x_2)-f(x_1)}{x_2-x_1}\).
  • \(b\): y-intercept in \(y = m x + b\).
  • Point-slope form: \(y - f(x_1) = m(x - x_1)\).
  • Two-point form: \(\dfrac{y - f(x_1)}{x - x_1} = \dfrac{f(x_2)-f(x_1)}{x_2-x_1}\).

How It Works: A Step-by-Step Example

Example. Let \(f(x)=x^3-2x+1\), \(x_1=0\), \(x_2=2\). Then:

\[ f(0)=1,\quad f(2)=2^3-2\cdot2+1=8-4+1=5 \] \[ m=\frac{5-1}{2-0}=2 \] \[ b=f(x_1)-m\,x_1 = 1-2\cdot 0 = 1 \] Secant line: \(y=2x+1\). Point-slope: \(y-1=2(x-0)\). Two-point: \(\frac{y-1}{x-0}=\frac{5-1}{2-0}\).

Frequently Asked Questions

Is the secant slope always constant on an interval?

Yes—the secant slope is a single number describing the net change of \(f\) over the two points you chose.

Can I approximate the tangent slope?

Use the point+step mode with a very small nonzero \(h\). The result approaches the derivative \(f'(x_0)\).

What if the function is not defined at a point?

The calculator flags evaluation errors. Choose points within the function’s domain, or modify the function.

Do I need radians or degrees?

By default, trig functions use radians (e.g., sin(x) expects radians). Convert degrees if needed.

How do I set a good chart range?

Use “Fit range” or manually set x-min/x-max so both secant intersection points are well within view.
