How to convert Hz to MHz

Hertz (Hz) and Megahertz (MHz) are both units of frequency. Hertz counts cycles per second, while Megahertz is a million cycles per second.

Conversion formulas

From Hertz to Megahertz:

\[ \text{MHz} = \frac{\text{Hz}}{1{,}000{,}000} \]

From Megahertz to Hertz:

\[ \text{Hz} = \text{MHz} \times 1{,}000{,}000 \]

Step-by-step example

Example 1: Convert 5,000,000 Hz to MHz.

  1. Write the formula: \(\text{MHz} = \text{Hz} \div 1{,}000{,}000\)
  2. Substitute the value: \(\text{MHz} = 5{,}000{,}000 \div 1{,}000{,}000\)
  3. Compute: \(\text{MHz} = 5\)

So, 5,000,000 Hz = 5 MHz.

Example 2: Convert 2.4 MHz to Hz.

  1. Use the reverse formula: \(\text{Hz} = \text{MHz} \times 1{,}000{,}000\)
  2. Substitute: \(\text{Hz} = 2.4 \times 1{,}000{,}000\)
  3. Compute: \(\text{Hz} = 2{,}400{,}000\)

So, 2.4 MHz = 2,400,000 Hz.

Scientific notation

For very large or very small frequencies, scientific notation is convenient:

  • 1 MHz = \(1 \times 10^{6}\) Hz
  • 100 MHz = \(1 \times 10^{8}\) Hz
  • 2.4 GHz = \(2.4 \times 10^{9}\) Hz = 2,400 MHz

This converter accepts inputs like 3e8, 4.5E6, or 1.2e7 directly.

Hz vs kHz vs MHz vs GHz

All of these are powers-of-ten multiples of Hertz:

  • 1 kHz = \(10^3\) Hz = 1,000 Hz
  • 1 MHz = \(10^6\) Hz = 1,000,000 Hz
  • 1 GHz = \(10^9\) Hz = 1,000,000,000 Hz

Moving one step up (Hz → kHz → MHz → GHz) divides by 1,000 each time. Moving one step down multiplies by 1,000.

Frequently asked questions

How do you convert Hz to MHz?

Divide the Hertz value by 1,000,000. For example, 20,000,000 Hz ÷ 1,000,000 = 20 MHz. The calculator does this automatically as you type.

How many Hz are in 1 MHz?

Exactly 1,000,000 Hz. That’s why MHz is convenient for radio, electronics, and CPU clock speeds: it avoids writing long strings of zeros.

Can I convert MHz back to Hz?

Yes. Enter a value in the MHz field and the Hz field updates instantly using \(\text{Hz} = \text{MHz} \times 1{,}000{,}000\).

Does rounding matter?

For most practical uses (electronics, RF, networking), rounding to 3–6 decimal places in MHz is more than enough. This tool keeps full internal precision and only rounds the displayed value.
