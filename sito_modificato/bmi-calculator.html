What is a set?

In elementary set theory, a set is a collection of distinct objects considered as a single entity. The objects are called elements or members of the set. For finite sets, we usually list the elements between curly braces:

\[ A = \{1, 2, 3\}, \qquad B = \{\text{red}, \text{blue}, \text{green}\}. \]

By definition, sets do not care about order or multiplicity: \(\{1,2,3\} = \{3,2,1\}\) and \(\{1,1,2,2,3\} = \{1,2,3\}\). The calculator adopts this convention by removing duplicates and internally sorting elements in a stable way.

Basic set operations

Given sets \(A\) and \(B\) inside some universal set \(U\), common operations include:

  • Union: elements in at least one of the sets \[ A \cup B = \{x : x \in A \text{ or } x \in B\}. \]
  • Intersection: elements common to both sets \[ A \cap B = \{x : x \in A \text{ and } x \in B\}. \]
  • Difference: elements in one set but not the other \[ A \setminus B = \{x : x \in A \text{ and } x \notin B\}. \]
  • Symmetric difference: elements in exactly one of the sets \[ A \triangle B = (A \setminus B) \cup (B \setminus A). \]
  • Complement (relative to \(U\)): \[ A^c = U \setminus A = \{x : x \in U \text{ and } x \notin A\}. \]

Subsets and equality

Set \(A\) is a subset of \(B\), written \(A \subseteq B\), if every element of \(A\) is also an element of \(B\). If \(A \subseteq B\) and \(B \subseteq A\), then the sets have exactly the same elements and are equal: \(A = B\).

The calculator checks subset relations like \(A \subseteq B\), \(B \subseteq U\), and equality relations such as \(A = B\), reporting them with clear symbolic notation so you can compare them to your own proofs or textbook exercises.

Three sets and Venn-style reasoning

With three sets \(A\), \(B\), and \(C\), we can study regions like the triple intersection \(A \cap B \cap C\) or mixed differences such as \(A \cap B \setminus C\). Rather than drawing a full Venn diagram, the calculator reports each relevant intersection and difference as an explicit set of elements.

Universal set and complements

Many problems in probability and logic work inside a fixed universal set \(U\), which contains all objects under consideration. Complements are always defined relative to \(U\): if an element is not in \(U\), it is simply outside the discussion.

When you provide \(U\), the calculator:

  • checks that \(A\), \(B\), and \(C\) are subsets of \(U\),
  • computes complements \(A^c\), \(B^c\), and \(C^c\),
  • flags any elements that appear in a set but not in the universal set.
