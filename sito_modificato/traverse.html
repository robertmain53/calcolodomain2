What this traverse calculator does

This tool is designed for land surveyors, civil engineers, and students who need to compute and adjust a traverse from measured distances and bearings.

It allows you to:

  • Enter a sequence of traverse legs with distance and bearing (decimal or DMS).
  • Compute departures (ΔE) and latitudes (ΔN) for each leg.
  • Build raw coordinates for each station from a starting point.
  • Evaluate linear misclosure and relative precision.
  • Apply Bowditch (compass) rule to distribute misclosure and obtain adjusted coordinates.
  • Compute the enclosed area of a closed traverse.

Key formulas used

1. Bearing to azimuth

Internally, all directions are converted to azimuths measured clockwise from North:

Quadrant bearings (e.g. N θ E):

  • N θ E → azimuth = θ
  • S θ E → azimuth = 180° − θ
  • S θ W → azimuth = 180° + θ
  • N θ W → azimuth = 360° − θ

2. Departure and latitude

For each leg with distance \(L\) and azimuth \( \alpha \) (degrees):

\[ \Delta E = L \sin(\alpha), \qquad \Delta N = L \cos(\alpha) \]

3. Coordinate accumulation

Starting from \((E_0, N_0)\), the coordinates of point \(i\) are:

\[ E_i = E_{i-1} + \Delta E_i, \qquad N_i = N_{i-1} + \Delta N_i \]

4. Misclosure and relative precision

For a closed traverse, the misclosure components are:

\[ \Delta E_c = \sum \Delta E_i, \qquad \Delta N_c = \sum \Delta N_i \]

Linear misclosure:

\[ f = \sqrt{\Delta E_c^2 + \Delta N_c^2} \]

Relative precision:

\[ \text{Relative precision} = \frac{\text{Total traverse length}}{f} \]

5. Bowditch (compass) rule

The Bowditch rule distributes the misclosure to each leg in proportion to its length. For leg \(i\) with length \(L_i\) and total length \(L_T\):

\[ \delta E_i = -\Delta E_c \cdot \frac{L_i}{L_T}, \qquad \delta N_i = -\Delta N_c \cdot \frac{L_i}{L_T} \]

Adjusted departures and latitudes:

\[ \Delta E_i' = \Delta E_i + \delta E_i, \qquad \Delta N_i' = \Delta N_i + \delta N_i \]

6. Traverse area (shoelace formula)

For a closed polygon with vertices \((E_i, N_i)\), \(i = 0, \dots, n-1\):

\[ A = \frac{1}{2} \left| \sum_{i=0}^{n-1} (E_i N_{i+1} - E_{i+1} N_i) \right| \]

where index \(n\) wraps back to 0.

Tips for reliable results

  • Use consistent units for all distances (e.g. meters or feet).
  • Ensure bearings are measured from the same reference (true, grid, or magnetic north).
  • For a closed traverse, check that the first and last points represent the same physical station.
  • Inspect the reported relative precision; values like 1:5,000 or better are typical for control surveys.

FAQ

Can I use this for open traverses?

Yes. Leave “Treat as closed traverse” unchecked. The calculator will still compute coordinates and report the vector from start to end, but it will not attempt to force the end point to coincide with the start.

Does the calculator correct angles?

No. This tool assumes that bearings are already adjusted or acceptable. It adjusts only the coordinate misclosure using the Bowditch rule. Angle adjustment methods (e.g. transit rule) are beyond the scope of this simple tool.

What coordinate system should I use?

You can work in any planar coordinate system: local grid, state plane, UTM, etc. Just keep units and reference direction consistent across all legs.
