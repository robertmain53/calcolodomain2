The Formula Explained

Transfer Function: \( H(s) = C(sI - A)^{-1}B + D \)

Glossary of Variables

  • A: State matrix
  • B: Input matrix
  • C: Output matrix
  • D: Feedthrough (or direct transmission) matrix

FAQs

What is a transfer function?

A transfer function expresses the relationship between the input and output of a system in the Laplace transform domain.

How do I enter a matrix?

Matrices should be entered row by row, with elements separated by spaces and rows separated by semicolons.

Why use this converter?

This converter simplifies the process of transforming complex state-space models into more intuitive transfer functions, essential for control system analysis.
