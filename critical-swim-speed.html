The Formula Explained

CSS = \(\frac{\text{Distance}}{\text{Time}}\)

Glossary of Variables

  • Distance: The swim distance measured in meters.
  • Time: The time taken to swim the distance in minutes and seconds.

How It Works: A Step-by-Step Example

For a 400m swim completed in 5:20 (five minutes and twenty seconds), the CSS is calculated as:

CSS = 400 / 320 = 1.25 m/s

Frequently Asked Questions (FAQ)

What is Critical Swim Speed (CSS)?

CSS is a measure of the pace you can maintain over a long distance. It's often used to define training intensities.

How accurate is this calculator?

The calculator provides an estimate based on input data. For precise results, ensure accurate distance and time measurements.

Can CSS vary between individuals?

Yes, CSS can vary based on individual fitness levels and swimming techniques.

Why should I know my CSS?

Knowing your CSS can help tailor your training regime for optimal improvements in endurance and speed.

Is CSS applicable to open water swimming?

CSS is primarily used for pool swimming but can also provide insights for open water training.
