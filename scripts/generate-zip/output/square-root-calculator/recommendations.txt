### Topic: Square Root Calculator (`square-root-calculator`)

**User Persona & Goal:** Anyone needing to find the square root of a number.

**Critique of Existing Pages:** This is a basic function on any calculator. To be better, a webpage must provide more educational value.

**Recommendations for `square-root-calculator.html`:**
*   **Simplifying Radicals:** For non-perfect squares (e.g., sqrt(50)), the calculator should provide both the decimal approximation (7.071...) and the simplified radical form (5√2). This is a key skill in algebra.
*   **Explain the Simplification:** Show the steps for simplifying: `sqrt(50) = sqrt(25 * 2) = sqrt(25) * sqrt(2) = 5√2`.
*   **Imaginary Numbers:** Correctly handle negative inputs by introducing imaginary numbers (e.g., `sqrt(-9) = 3i`).
