### Topic: Mode Calculator (`mode-calculator`)

**User Persona & Goal:** A student learning basic statistics who needs to find the most frequently occurring number in a data set.

**Critique of Existing Pages:** Most tools find the mode but don't handle multi-modal (multiple modes) or no-mode scenarios gracefully.

**Recommendations for `mode-calculator.html`:**
*   **Simple Data Input:** A text area where a user can paste a list of numbers.
*   **Clear Results:** The output should explicitly state the mode(s). It should also be able to report "No mode found" or list all modes if the set is multi-modal (e.g., "Modes: 4, 7").
*   **Frequency Table:** Display a frequency table showing how many times each number appeared in the set. This shows the "work" behind finding the mode and provides more insight into the data distribution.
*   **Mean, Median, Mode:** Offer to calculate the mean and median as well, as these three measures of central tendency are almost always taught together.
