### Topic: Circle Calculator (`circle-calculator`)

**User Persona & Goal:** A student or DIYer needing to find the properties of a circle from one known measurement.

**Critique of Existing Pages:** Very common calculator. To be better, it must be an "all-in-one" tool.

**Recommendations for `circle-calculator.html`:**
*   **Input Any Property:** Allow the user to input either the radius, diameter, circumference, or area.
*   **Calculate Everything Else:** Based on the single input, calculate all the other three properties. For example, if the user enters the area, the calculator should solve for the radius and then use that to find the diameter and circumference.
*   **Show the Formulas:** For each calculated value, show the formula that was used (e.g., `C = 2πr`, `A = πr^2`).
