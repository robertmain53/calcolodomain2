### Topic: Matrix Calculator (`matrix-calculator`)

**User Persona & Goal:** A student or engineer working with linear algebra. They need to perform operations like addition, multiplication, finding the determinant, or inverting a matrix.

**Critique of Existing Pages:** Many matrix calculators have clunky interfaces for inputting matrix values and may only perform one operation at a time.

**Recommendations for `matrix-calculator.html`:**
*   **Flexible Matrix Input:** Allow users to easily define matrix dimensions (e.g., 2x3, 4x4) and input values in a clean grid. Support copy/pasting from spreadsheets.
*   **All-in-One Analysis:** After a user inputs a matrix, calculate and display multiple properties at once: determinant, trace, inverse, transpose, and eigenvalues/eigenvectors. This provides a complete picture without re-entering data.
*   **Step-by-Step Solutions:** For educational purposes, offer an option to show the step-by-step work for operations like finding the inverse or determinant. This is a huge value-add over just giving the answer.
*   **Binary Operations:** Support operations between two matrices (A+B, A*B) with clear error messaging for incompatible dimensions.
