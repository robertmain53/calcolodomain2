### Topic: Irregular Polygon Calculator (`irregular-polygon-calculator`)

**User Persona & Goal:** A surveyor, landowner, or advanced student needing to find the area of a polygon that is not regular.

**Critique of Existing Pages:** This is a complex problem that most calculators don't even attempt.

**Recommendations for `irregular-polygon-calculator.html`:**
*   **Coordinate Input:** The user should be able to input a list of the polygon's vertices (x,y coordinates) in order around the perimeter.
*   **Shoelace Formula:** Implement the Shoelace (or Surveyor's) Formula to calculate the area. This is the standard algorithm for this problem.
*   **Show the Work:** Display the matrix-like calculation of the Shoelace formula. This is essential for trusting the result and for learning the method.
*   **Visual Plot:** Plot the vertices and draw the polygon on a graph so the user can confirm they entered the coordinates correctly.
