### Topic: Function Calculator (`function-calculator`)

**User Persona & Goal:** A high school or college student studying algebra or calculus. They need to evaluate a function at a specific point.

**Critique of Existing Pages:** Basic evaluators exist, but they often lack graphical context and don't support a wide range of mathematical functions.

**Recommendations for `function-calculator.html`:**
*   **Interactive Input:** Allow the user to input a function `f(x)` (e.g., `x^2 + sin(x)`) and a value for `x`.
*   **Live Graphing:** As the user types the function, generate a plot of it. When they evaluate it at a point `x`, mark that point clearly on the graph. This connects the abstract formula to a visual representation.
*   **Function Library:** Support a wide range of functions: trigonometric, logarithmic, exponential, etc., and provide clear examples of the syntax to use (e.g., `log(x)`, `exp(x)`).
*   **Table of Values:** Automatically generate a table of values for the function around the evaluated point to show the function's behavior.
