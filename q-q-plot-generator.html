The Formula Explained

The Q-Q plot compares the quantiles of the data against the quantiles of the distribution. For a perfect match, the points should lie on a straight line.

Glossary of Terms

  • Quantile: A point in your data below which a certain percentage falls.
  • Distribution: A statistical function that describes all the possible values and likelihoods that a random variable can take.

How It Works: A Step-by-Step Example

Enter your data and select the distribution you want to compare it against. The tool will plot your data's quantiles against the theoretical quantiles of the selected distribution.

Frequently Asked Questions (FAQ)

What is a Q-Q plot?

A Q-Q plot is a graphical tool to help assess if a data set follows a specific distribution.

How do I interpret a Q-Q plot?

If the points on the Q-Q plot lie on a straight line, the data likely follows the specified distribution.

What distributions can I compare against?

This tool supports Normal, Exponential, and Uniform distributions.
